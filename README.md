# ARTS Benchmark

`apps` and `ocr` directories are from XSOCR project.
We developed OCR based on ARTS and connected several benchmarks to ARTS-OCR.
You can find license information of OCR and OCR-based applications in each directory.
Here are current applications supported in this repository:

- basicIO
- cholesky
- cholesky_mkl
- CoMD
- fft
- fibonacci
- globalsum_noshim
- globalsum_pcg
- globalsum_shim
- hpcg
- hpgmg
- npb-cg
- p2p
- quicksort
- reduction
- sar_huge
- sar_large
- sar_medium
- sar_problem_size_scaling
- ocr_small
- ocr_tiny
- skel (simple printf)
- smithwaterman
- Stencil1D_chandra
- Stencil1D_david_guid
- Stencil1D_david_guidPI
- Stencil1D_david_once
- Stencil1D_david_oncePI
- Stencil1D_david_sticky
- Stencil1D_david_stickyLG (not supported)
- Stencil2D
- triangle
- XSBench (not supported)
